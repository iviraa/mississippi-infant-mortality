# Processed data

Exported contents of the analytical schema. Regenerated by `make export` (or as part of `make all`). Each file corresponds to a table in the PostgreSQL `public` schema.

- `schema.sql`: full DDL for the analytical tables.
- `dim_*.csv`: dimension tables (geography, measure, facility, year).
- `fact_*.csv`: fact tables (PLACES, SVI long, SVI wide, ACS, IMR, hospital quality, HPSA).
- `mart_*.csv`: the five analytical marts (MRI, drive time, double burden, top 20 priority, HRRP regressivity).
- `svi_county.csv` and `stg_msdh_imr_race.csv`: supporting tables read by the marts and the Q15 disparity query.

Geometry columns are omitted from the CSVs because their binary representations are not human readable. The tract and county polygons live in `dim_geography.geom` and are reproducible by running `make extract && make load`.
